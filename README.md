This is a fork of https://keepass.info that adds support for .NET Standard and provides a NuGet package for easy consumption. The current code is based off of KeePass 2.34 and licensed according to the license of KeePass (see [License](docs/License.txt)).

This is available via the MyGet feed: `https://www.myget.org/F/keepass/api/v3/index.json`
